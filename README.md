## Como subir no GitHub/Vercel

1. Apague todos os arquivos antigos do repositório.
2. Extraia este ZIP.
3. Suba todos os arquivos e pastas extraídos diretamente na raiz do GitHub.
4. A raiz do GitHub deve ter index.html, app.js, styles.css, firebase-config.js, vercel.json, README.md e a pasta assets.
5. Aguarde o deploy automático do Vercel ou clique em Redeploy.
6. Abra em janela anônima ou use Ctrl + F5.
